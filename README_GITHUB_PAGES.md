
# RenomaPro — Frontend for GitHub Pages

This folder contains a static frontend version ready for GitHub Pages.

### Deployment:

1. Go to GitHub → Settings → Pages
2. Select:
   - Source: Deploy from branch
   - Branch: main
   - Folder: /docs
3. Save.
4. Page will be available at:
   https://twoj-login.github.io/renomapro/

### Contains:
- index.html with logo + favicon
- assets/logo.png
- assets/favicon.png
- all frontend pages including legal, reviews, client/admin/owner panels
